# 要件

## 概要

BitCraftというMMOで、有志が作成したMapのURLを生成するツールを作成する。将来的にはMapを直接サイトに埋め込み表示したい。

## 仕様

- Typescript
- Astro (ほぼbuild用)
- SolidJS

- TailwindCSSを使用しない

ベースURL -> <https://bitcraftmap.com/>

### string

`enemyId`

-> `https://bitcraftmap.com/?enemyId=xxxxxxxxx`

`resourceId`

-> `https://bitcraftmap.com/?resourceId=xxxxxxxxx`

## UI

丸みが無い角ばったデザインで、シンプルなUIを目指す。

リソース選択は表形式で行う。
表の行はリソースの種類、列はティアを表す。
表のセルを選択可能にし、選択したリソースとティアの組み合わせをurlとして出力する。
セルをクリックするたびにURLが更新されるようにする。

ユニークリソースの扱いについては、リソースの表の下に別途表を設け、そこにユニークリソースとモンスターをまとめて表示する。

種類の並びについては、上から Flower->Mushroom->Berry->Fiber_Plant->Ore_Vein->Tree->Rock->Glyph->Sand->Clay,Sailing->Bait_Fish->Lake_ Fish->Ocean_Fish->Animal, Unique_Resource->Ancient->Denの順で行う。

i18nは考慮せず英語表記で行う。
